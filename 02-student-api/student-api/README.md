# Student API

A simple REST API built with Spring Boot.

## Features

- Get all students
- Get student by ID
- Add a new student
- Update student details
- Delete a student

## Technologies

- Java 21
- Spring Boot 3.5
- Maven
- Postman

## Endpoints

GET /students

GET /students/{id}

POST /students

PUT /students/{id}

DELETE /students/{id}
